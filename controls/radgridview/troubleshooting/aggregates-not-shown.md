# Aggregates are not shown

__PROBLEM__

AggregateResults are not shown at the bottom of the RadGridView.

__CAUSE__

By default the RadGridView does not show the column footers. 

__SOLUTION__

Set the __ShowColumnFooters__ property to __"True"__.
